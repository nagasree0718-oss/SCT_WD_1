# AI Learning Hub

## SkillCraft Technology - Web Development Internship

### Task 01: Responsive Landing Page

## Project Overview

AI Learning Hub is a modern and responsive landing page developed as part of the SkillCraft Technology Web Development Internship Task 01. The project is designed to provide an engaging and user-friendly platform for individuals interested in learning Artificial Intelligence and related technologies.

The primary objective of this task was to create an interactive navigation menu that remains fixed across all pages and changes its appearance when users scroll or interact with menu items. The website combines modern design principles with responsive layouts to deliver a seamless browsing experience across different devices.

## Features

* Fixed Interactive Navigation Menu
* Dynamic Navbar Style Change on Scroll
* Hover Effects on Navigation Items
* Responsive Design for Mobile, Tablet, and Desktop
* Modern AI-Themed User Interface
* Attractive Hero Section
* AI Courses Showcase
* Popular AI Tools Section
* Learning Roadmap Section
* AI Project Showcase
* Smooth Scrolling Navigation
* Mobile-Friendly Navigation Menu
* Glassmorphism Card Design
* Smooth Animations and Transitions
* Professional Layout and User Experience
